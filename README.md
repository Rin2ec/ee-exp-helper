# 伊伊經驗小幫手

🎮 《**Artale**》EXP 統計工具，透過即時 OCR 擷取畫面，自動分析經驗效率，並提供浮動 HUD 與多項統計功能。

![screenshot](https://truth.bahamut.com.tw/s01/202508/forum/79354/40b9d4c7b1d05ac2123dadc95e4d4457.JPG)


---

## ✨ 功能特色
- 📷 擷取指定應用程式畫面，自由框選 EXP 區塊，不受其他視窗遮擋
- 🔍 即時 OCR 辨識畫面 EXP 數字
- 💬 即時系統訊息：操作提示、錯誤資訊、OCR 狀態等會即時顯示於畫面
- 🛠️ 快捷鍵選單：自訂各項功能快捷鍵
- 📈 實時統計：
  - 當前 EXP
  - 每分鐘 EXP
  - 預估 10 分鐘 EXP  
    🟢 統計時長 ≥ 10 分鐘  
    🟡 統計時長 ≥ 3 分鐘  
    🔴 統計時長不足 3 分鐘 
  - 預估 60 分鐘 EXP
  - 10分鐘實際累積 EXP
  - 60分鐘實際累積 EXP
  - 總累積 EXP
  - 預估1/10/60分鐘經驗百分比
  - 升級預估時間  
  - 當前統計時間
  - OCR 準確率
    🟢 準確率 ≥ 90%  
    🟡 準確率 ≥ 60%  
    🔴 準確率 ≥ 0%
- 🎮 **遊戲模式浮動 HUD：**
  - 支援自由拖曳與縮放 
    縮放調整方式: 點選視窗頂部拉桿進行調整
  - 支援透明度調整 
    透明度調整方式: 點選視窗頂右側拉桿進行調整
  - 快速顯示 / 隱藏
  - 全部欄位可即時開關
  - 欄位顯示可即時開關
  - 進度條模式切換（時間）
  - 顯示／隱藏即時 EXP 動態圖表
  - 支援 OBS 實況視窗擷取
  - 簡化模式切換
- ⌨️ **快捷鍵支援**：
  - `F7`：開始/暫停/繼續統計  
  - `F8`：重置統計  
  - `F9`：切換遊戲模式（HUD 開/關）
  - `F10`：截圖遊戲模式統計資料
- 💾 自動儲存設定（偵測區域、顯示模式、視窗位置、視窗大小、透明度與快捷鍵設定）
- 📤 匯出功能：可將統計資料另存為 `.csv` 檔案
- 🛡️ 使用外部畫面偵測，純本地執行、不連網不上傳任何資料，零風險安心使用
- 💡 效能經過優化，適合長時間穩定使用

## 🚀 快速開始

1. 下載並解壓縮
2. 執行應用程式 `伊伊經驗小幫手.exe`  
   💡如果跳「Windows已保護您的電腦」為正常，請點選「其他資訊」後按「仍要執行」啟動
   💡如果開啟無反應或擷取黑畫面請用系統管理員執行
3. 點擊「選取視窗」選擇指定視窗 (預設自動帶入 Artale)
4. 點擊「選取範圍」框選 EXP 數值區塊  
   ⚠️ **請記得右邊留點空間（大約 2～3 位數）**，避免從 `99` 升到 `100` 時被裁掉，影響偵測。
5. 按下「開始」或「 F7 」 開始統計
6. 按下「遊戲模式」或「 F9 」 進入遊戲模式顯示浮動 HUD  
   🔍遊戲模式 UI 縮放方式：使用最上方的拉桿即可調整視窗大小

## ❓ 常見問題 QA

### 1️⃣ 無法下載
**原因：**  
部分防毒軟體可能誤判為未知程式並阻擋下載。

**解法：**  
1. 開啟 Windows 搜尋「病毒與威脅防護」  
2. 點選「保護的歷程記錄」  
3. 將「伊伊經驗小幫手」設為允許

---

### 2️⃣ 無法開啟程式
**原因：**  
Windows SmartScreen 阻擋應用程式執行。

**解法：**  
1. 點選「更多資訊」  
2. 點選「仍要執行」

---

### 3️⃣ 開啟失敗或選取視窗後黑畫面
**原因：**  
未以系統管理員身分執行，導致擷取畫面失敗。

**解法：**  
僅需設定一次：  
1. 右鍵點選 `伊伊經驗小幫手.exe` → 內容  
2. 切換到「相容性」分頁  
3. 勾選「以系統管理員身分執行此程式」  
4. 確定後重新啟動程式

---

### 4️⃣ 小幫手介面視窗模糊
**原因：**  
Windows DPI 縮放導致畫面顯示不清晰。

**解法：**  
1. 右鍵點選程式 → 內容  
2. 切換到「相容性」分頁  
3. 點選「更改高 DPI 設定」  
4. 勾選「覆寫高 DPI 縮放行為」，選擇「應用程式」

---

### 5️⃣ 選取視窗列表中沒有 Artale
**原因：**  
遊戲視窗處於最小化狀態，無法偵測。

**解法：**  
1. 確認 Artale 已開啟且非最小化  
2. 回到小幫手 → 點擊「重新整理視窗列表」  
3. 再次選取 Artale

💡 若已選取過視窗但啟動後畫面為黑：  
→ 請取消最小化遊戲視窗，重新啟動小幫手即可。

---

### 6️⃣ 選取視窗後畫面過白導致辨識失敗
**原因：**  
HDR 模式造成畫面亮度異常。

**解法：**  
請關閉顯示器的 HDR 模式後再次辨識。

---

### 7️⃣ 辨識經常失敗或 EXP 數值跳動
**原因：**  
選取範圍異常、OCR 區域解析度不足。

**解法：**  
1. 重新選取偵測範圍  
2. 若仍有問題，可調整 `config.json` 中的解析度設定（主視窗右上角 → 開啟設定檔資料夾）：

<pre><code>{
  "selection": {
    "x": 100,
    "y": 100,
    "width": 400,
    "height": 100,
    "videoWidth": 1920,
    "videoHeight": 1080
  }
}
</code></pre>

- `videoWidth`：畫面寬度（預設 1920）  
- `videoHeight`：畫面高度（預設 1080）  

📌 建議不要超過 `2560x1440`，解析度越高越清晰，但會增加效能消耗。

---

### 8️⃣ 是否會被偵測為外掛或腳本？
**原因：**  
有玩家擔心程式會被誤判為外掛導致封號。

**回答：**  
不會。  
本程式僅透過 **畫面擷取 + OCR 辨識** 進行統計，不會修改遊戲、不發送封包、不模擬操作，屬於**單機輔助工具**，請安心使用。
